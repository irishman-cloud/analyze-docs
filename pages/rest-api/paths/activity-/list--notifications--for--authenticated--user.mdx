# GET - `/notifications`

List notifications for the authenticated user

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| all  | If `true`, show notifications marked as read. | boolean in query | `undefined` |
| participating  | If `true`, only shows notifications in which the user is directly participating or mentioned. | boolean in query | `undefined` |
| since  | Only show notifications updated after the given time. This is a timestamp in [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601) format: `YYYY-MM-DDTHH:MM:SSZ`. | string in query | `undefined` |
| before  | Only show notifications updated before the given time. This is a timestamp in [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601) format: `YYYY-MM-DDTHH:MM:SSZ`. | string in query | `undefined` |
| page  | Page number of the results to fetch. | integer in query | `undefined` |
| per_page  | The number of results per page (max 50). | integer in query | `undefined` |
