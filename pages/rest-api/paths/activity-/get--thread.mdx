# GET - `/notifications/threads/{thread_id}`

Get a thread

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| thread_id <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The unique identifier of the notification thread. This corresponds to the value returned in the `id` field when you retrieve notifications (for example with the [`GET /notifications` operation](https://docs.github.com/rest/reference/activity#list-notifications-for-the-authenticated-user)). | integer in path | `undefined` |
