# GET - `/users`

List users

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| since  | A user ID. Only return users with an ID greater than this ID. | integer in query | `undefined` |
| per_page  | The number of results per page (max 100). | integer in query | `undefined` |
