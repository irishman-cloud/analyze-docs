# GET - `/users/{username}/hovercard`

Get contextual information for a user

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| username <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The handle for the GitHub user account. | string in path | `undefined` |
| subject_type  | Identifies which additional information you'd like to receive about the person's hovercard. Can be `organization`, `repository`, `issue`, `pull_request`. **Required** when using `subject_id`. | string in query | `undefined` |
| subject_id  | Uses the ID for the `subject_type` you specified. **Required** when using `subject_type`. | string in query | `undefined` |
