# GET - `/user/gpg_keys/{gpg_key_id}`

Get a GPG key for the authenticated user

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| gpg_key_id <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The unique identifier of the GPG key. | integer in path | `undefined` |
