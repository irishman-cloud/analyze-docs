# GET - `/projects/{project_id}/collaborators/{username}/permission`

Get project permission for a user

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| project_id <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The unique identifier of the project. | integer in path | `undefined` |
| username <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The handle for the GitHub user account. | string in path | `undefined` |
