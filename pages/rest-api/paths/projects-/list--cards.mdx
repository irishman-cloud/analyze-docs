# GET - `/projects/columns/{column_id}/cards`

List project cards

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| column_id <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The unique identifier of the column. | integer in path | `undefined` |
| archived_state  | Filters the project cards that are returned by the card's state. | string in query | `undefined` |
| per_page  | The number of results per page (max 100). | integer in query | `undefined` |
| page  | Page number of the results to fetch. | integer in query | `undefined` |
