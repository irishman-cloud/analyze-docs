# GET - `/gists/{gist_id}/commits`

List gist commits

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| gist_id <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The unique identifier of the gist. | string in path | `undefined` |
| per_page  | The number of results per page (max 100). | integer in query | `undefined` |
| page  | Page number of the results to fetch. | integer in query | `undefined` |
