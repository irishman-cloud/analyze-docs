# GET - `/user/migrations/{migration_id}`

Get a user migration status

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| migration_id <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The unique identifier of the migration. | integer in path | `undefined` |
| exclude  |  | array in query | `undefined` |
