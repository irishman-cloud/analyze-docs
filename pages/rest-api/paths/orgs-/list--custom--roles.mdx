# GET - `/organizations/{organization_id}/custom_roles`

List custom repository roles in an organization

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| organization_id <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The unique identifier of the organization. | string in path | `undefined` |
