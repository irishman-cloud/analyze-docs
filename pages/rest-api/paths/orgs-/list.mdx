# GET - `/organizations`

List organizations

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| since  | An organization ID. Only return organizations with an ID greater than this ID. | integer in query | `undefined` |
| per_page  | The number of results per page (max 100). | integer in query | `undefined` |
