# GET - `/orgs/{org}/members`

List organization members

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| org <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The organization name. The name is not case sensitive. | string in path | `undefined` |
| filter  | Filter members returned in the list. `2fa_disabled` means that only members without [two-factor authentication](https://github.com/blog/1614-two-factor-authentication) enabled will be returned. This options is only available for organization owners. | string in query | `undefined` |
| role  | Filter members returned by their role. | string in query | `undefined` |
| per_page  | The number of results per page (max 100). | integer in query | `undefined` |
| page  | Page number of the results to fetch. | integer in query | `undefined` |
