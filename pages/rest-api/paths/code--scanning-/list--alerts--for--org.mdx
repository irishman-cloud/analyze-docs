# GET - `/orgs/{org}/code-scanning/alerts`

List code scanning alerts for an organization

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| org <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The organization name. The name is not case sensitive. | string in path | `undefined` |
| tool_name  | The name of a code scanning tool. Only results by this tool will be listed. You can specify the tool by using either `tool_name` or `tool_guid`, but not both. | string in query | `undefined` |
| tool_guid  | The GUID of a code scanning tool. Only results by this tool will be listed. Note that some code scanning tools may not include a GUID in their analysis data. You can specify the tool by using either `tool_guid` or `tool_name`, but not both. | string in query | `undefined` |
| before  | A cursor, as given in the [Link header](https://docs.github.com/rest/overview/resources-in-the-rest-api#link-header). If specified, the query only searches for results before this cursor. | string in query | `undefined` |
| after  | A cursor, as given in the [Link header](https://docs.github.com/rest/overview/resources-in-the-rest-api#link-header). If specified, the query only searches for results after this cursor. | string in query | `undefined` |
| page  | Page number of the results to fetch. | integer in query | `undefined` |
| per_page  | The number of results per page (max 100). | integer in query | `undefined` |
| direction  | The direction to sort the results by. | string in query | `undefined` |
| state  | If specified, only code scanning alerts with this state will be returned. | string in query | `undefined` |
| sort  | The property by which to sort the results. | string in query | `undefined` |
