Get a user installation for the authenticated app

| Name | Description | Type | Example |

| username <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The handle for the GitHub user account. | string in path | `undefined` |
