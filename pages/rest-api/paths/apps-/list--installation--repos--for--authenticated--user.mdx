# GET - `/user/installations/{installation_id}/repositories`

List repositories accessible to the user access token

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| installation_id <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The unique identifier of the installation. | integer in path | `undefined` |
| per_page  | The number of results per page (max 100). | integer in query | `undefined` |
| page  | Page number of the results to fetch. | integer in query | `undefined` |
