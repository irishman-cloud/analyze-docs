# GET - `/marketplace_listing/plans/{plan_id}/accounts`

List accounts for a plan

## Parameters

| Name | Description | Type | Example |
| :---- | :---- | :---- | :---- |
| plan_id <span class="bg-blue-100 text-blue-800 text-xs font-medium me-2 px-2.5 py-0.5 rounded dark:bg-blue-900 dark:text-blue-300">Required</span> | The unique identifier of the plan. | integer in path | `undefined` |
| sort  | The property to sort the results by. `created` means when the repository was starred. `updated` means when the repository was last pushed to. | string in query | `undefined` |
| direction  | To return the oldest accounts first, set to `asc`. Ignored without the `sort` parameter. | string in query | `undefined` |
| per_page  | The number of results per page (max 100). | integer in query | `undefined` |
| page  | Page number of the results to fetch. | integer in query | `undefined` |
